# Game Scan

## Requirements

- [Rust](https://www.rust-lang.org)
    - cargo >= v1.49.0
    - rustup >= v1.23.1
    - rustc >= v1.49.0
- [Node](https://nodejs.org)
    - node \>= 12.20.0
    - npm \>= 6.14.8
